# Qt_OCC CAD アプリケーション

Qt_OCCは、ユーザーインターフェースにQtフレームワーク、ジオメトリカーネルにOpenCASCADE Technology (OCCT) を使用して開発されたデスクトップCADアプリケーションです。3Dモデリングと視覚化のためのプラットフォームを提供することを目的としており、形状作成、操作、解析のためのさまざまな機能を組み込んでいます。

## ✨ 機能

### 📂 ファイル管理
*   **入出力**: 新規プロジェクト作成、既存ファイルのオープン、保存、モデルのエクスポートをサポート。

### 🎥 視覚化とナビゲーション
*   **標準ビュー**: 等角投影(Isometric)、上面、下面、左側面、右側面、前面、背面ビューを素早く切り替え。
*   **表示モード**: ワイヤーフレーム、シェーディング、エッジ付きシェーディングなどのレンダリングスタイルを切り替え。
*   **ビュー制御**: ビューをシーン全体に自動フィット。

### 🛠️ モデリングツール
*   **2D プリミティブ**: 点、線、長方形、円、円弧、楕円、ポリゴン、ベジェ曲線、NURBS曲線の作成。
*   **3D プリミティブ**: 直方体(Box)、球、円柱、円錐、角錐の生成。
*   **ブール演算**: ソリッドに対する和(Fuse)、積(Common)、差(Cut)演算の実行。
*   **変形・編集**:
    *   **ミラー**: 平面または軸によるオブジェクトの鏡像作成。
    *   **パターン**: 線形および円形パターンの作成。
    *   **シェル**: ソリッドをシェル（中空）構造に変換。

### 📏 計測と解析
*   **計測**: 距離、エッジ長、円弧長/半径、角度を測定する精密ツール。
*   **解析**: オブジェクト間の干渉チェック。
*   **アセンブリ可視化**: 内部コンポーネントを検査するための分解ビュー(Exploded View)。
*   **クリッピング**: 断面ビューを確認するためのクリッピング平面の適用。

### ⚙️ ユーティリティとカスタマイズ
*   **トランスフォーム**: オブジェクトの移動、回転、拡大縮小を行うインタラクティブツール。
*   **作業平面**: 設定可能な座標系ヘルパー。
*   **選択フィルタ**: 頂点、エッジ、面、ソリッドによる選択フィルタリング。
*   **テーマ対応**: ライトテーマとダークテーマの切り替え。
*   **ローカライズ**: 多言語サポート。

## 🛠️ 技術スタック

*   **C++**: アプリケーションのコアロジックは最新のC++ (C++17標準) で記述されています。
*   **Qt Framework**: GUI全体に使用され、SARibbonを使用してモダンなリボンスタイルのツールバーを実現しています。
*   **OpenCASCADE Technology (OCCT)**: すべてのCAD操作を処理する強力なオープンソース3D幾何形状モデリングカーネル。
*   **CMake**: プロジェクトの構成とビルドに使用されるクロスプラットフォームビルドシステム。

## 🚀 はじめに

以下の手順に従って、開発およびテスト目的でローカルマシン上にプロジェクトのコピーをセットアップしてください。

### 前提条件

システムに以下のソフトウェアがインストールされている必要があります。

*   **C++17準拠のコンパイラ**:
    *   Windows: MSVC 2019 以降
*   **CMake**: バージョン 3.16 以上。
*   **Qt Framework**: バージョン 6.2 以上。コンパイラに一致するコンポーネントをインストールしてください。
*   **OpenCASCADE Technology (OCCT)**: バージョン 7.6.0 以上。

### コンパイル手順

1.  **リポジトリのクローン**

    ```bash
    git clone https://github.com/bezierC0/Qt_OCC.git
    cd Qt_OCC
    ```

2.  **ビルドディレクトリの作成**

    ```bash
    mkdir build
    cd build
    ```

3.  **CMakeでの構成**

    QtおよびOCCTのインストールパスを正しく設定してください。

    ```bash
    # Windows Visual Studio の例
    cmake .. -G "Visual Studio 17 2019" -A x64 \
    -DQt5_DIR="C:/Qt/5.15.2/msvc2019_64/lib/cmake/Qt5" \
    -DOpenCASCADE_DIR="C:/Libs/OCCT/occt-7.8.0-vc14-64" \
    -DOpenCASCADE_3RDPARTY_DIR="C:/Libs/OCCT/3rdparty-vc14-64"
    ```

4.  **プロジェクトのビルド**

    ```bash
    cmake --build . --config Release
    ```

5.  **アプリケーションの実行**

    実行ファイルは `build/bin/Release` (Windows) または `build/bin` (Linux) ディレクトリに生成されます。
